Bootstrap-komponenttien pikamuistitaulukko

Komponentti Mihin sopii Tärkeät luokat / attribuutit Huomiot
Grid (Row/Col) Sivun peruslayout, sarakkeet .container, .row, .col-*, .g-* Mobile‑first; käytä col-md-6 jne. taukoihin
Navbar Ylävalikko / navigaatio .navbar, .navbar-expand-*, .navbar-dark/.navbar-light, .bg-* Offcanvas/collapse mobiiliin; vältä toista navbaria footerissa
Accordion Kollapsoituvat osiot (kuten vitsit) .accordion, .accordion-item, data-bs-target Yksi auki kerrallaan: käytä data-bs-parent
Cards Kuva + otsikko + teksti -kortit .card, .card-body, .card-img-top Ryhmittelyyn .row .col tai .card-group
Buttons Toimintapainikkeet .btn, .btn-primary, .btn-outline-*, .btn-sm Lisää ikonit esim. Bootstrap Iconsilla
Alerts Ilmoitukset / huomautukset .alert + .alert-* (success, warning…) Dismiss: .alert-dismissible + close‑button
Modal Ponnahdusikkunat .modal, data-bs-toggle="modal", data-bs-target Muista modal-dialog-centered tarvittaessa
Offcanvas Sivuvalikko / mobiilivalikko .offcanvas, data-bs-target, .offcanvas-end Hyvä korvaaja mobiili‑navbarin collapselle
Toast Pienet ilmoitukset kulmassa .toast, JS‑init Vaatii JS‑alustuksen; pinoa .toast-containeriin
Badges Pienet laskurit / tagit .badge, .text-bg-* Lisää esim. napin sisään
Tables Listaukset / data .table, .table-striped, .table-hover, .table-responsive Responsiivisuus: kääri div.table-responsiveiin
Forms Syötteet ja lomakkeet .form-control, .form-check, .input-group Käytä .row .col ryhmittelyyn
Vinkki: pienillä näytöillä taulukko vierittyy vaakasuunnassa.
